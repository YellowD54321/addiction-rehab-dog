# 網站主要功能

## 訂下約定

- 使用者可以在進入網站時，選擇想要戒除的癮頭。目前以 social media 為主。
  - instagram reels
  - facebook 短影片
  - youtube shorts
  - threads
  - X
  - reddit
  - ptt
- 選擇好產品後，跟自己訂下約定。例如：
  - 我在今天完全不開啟 ig 滑短影音
  - 我今天只會開啟 1 個小時的 ig
  - 我今天會比昨天減少 0.5 小時的手機使用時間

## 遵守約定

- 訂定完約定後，畫面上不會再顯示訂下約定的表格，而是兩個按鈕：成功的「I made it!」和失敗的「I didn't made it...」
- 按下成功之後，會面會出現恭喜的圖示。
- 按下失敗之後，畫面會出現沮喪的圖示。

## 資料格式

- 約定項目(ig reels, yt shorts 等等)是寫死在程式碼的固定項目。目前只有這幾項，以後會再擴充。
- 約定內容是單純文字，會有個 input 可以讓使用者自行輸入。
- 送出約定時，資料會被儲存在 indexdb。
- 遵守約定成功或失敗也會把資料寫在 indexdb。
- 現階段沒有後端，未來會新增後端才能讓資料儲存在伺服器。
- 瀏覽器的 indexdb 資料可以在未來顯示為圖表格式，讓使用者看到他哪幾天成功、哪幾天失敗、連續成功了幾天。（圖表顯示不在這次範圍內，只是未來需要支援。）
